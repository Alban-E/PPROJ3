# PPROJ3: SUPIFY

## Instructions de lancement de l'application web: 

1. Cloner le repo.
2. Lancer cette commande à la racine du repo :

```bash
    docker compose up
```

3. Acceder à l'application via l'url suivante :

```
    http://localhost:${FRONTEND_PORT}$/
```

## Choix techniques

* ***Base de donnée: Mongo***
* ***Backend: Express js***
* ***Frontend: React***
