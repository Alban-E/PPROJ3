# PPROJ3: SUPIFY

## Instructions de lancement de l'application web: 

Si docker est intallé et fonctionnel :

1. Cloner le repo : 
```
    https://github.com/Alban-E/PPROJ3
```

2. Lancer docker puis executer cette commande à la racine du repo :

```bash
    docker compose up
```

3. Définir les variables d'environnement (voir documentation technique)

4. Acceder à l'application via l'url suivante :

```
    http://localhost:${FRONTEND_PORT}$/
```
